Rough Life

Inspired by Conway's Game of Life
Instructions: Rough Life is based on Conway's classic game of life.
    Rules: The grid is made up of squares. Each step is determined by how many live squares are surrounding it
  • 1) If a square is alive, and it has 2 or 3 live squares around it, it lives. Otherwise, it dies.
  • 2) If a square is dead, and it has exactly 3 live squares around it, it resurrects.
  • Start begins the game
  • Rainbow Random Colors Follow Path and Fill Path change the users view of the grid
  • Change Living Percentage changes what percentage of squares begin living
  • Place object Fill Board and Random Objects places an amount of conway-type object
  • Step carries through one iteration at a time
